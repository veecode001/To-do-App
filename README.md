# My Task Manager

A task management web app built with React. Add, edit, complete, and delete tasks — with deadline reminders that alert you when time's up.

**Live demo:** https://task-manager-veecode.vercel.app/

## Features

- Add tasks instantly (press Enter or click Add Task)
- Edit existing tasks
- Mark tasks as complete with a celebration animation
- Delete tasks that are no longer needed
- Set deadlines with both date and time
- Get a sound alert and browser notification when a deadline is reached
- Fully responsive — works on both desktop and mobile

## Tech Stack

- React
- CSS (custom styling, no framework)
- Deployed on Vercel
- Version controlled with Git & GitHub

## Run Locally

Clone the repository:

\`\`\`bash
git clone https://github.com/veecode001/To-do-App.git
\`\`\`

Navigate into the project folder:

\`\`\`bash
cd To-do-App
\`\`\`

Install dependencies:

\`\`\`bash
npm install
\`\`\`

Start the development server:

\`\`\`bash
npm start
\`\`\`

The app will open at `http://localhost:3000`
